1. Componente Dropdown

Elemento desplegable con una lista de opciones que se muestran al hacer click sobre el elemento padre.

Ejemplo:

Presentación
Curriculum vitae

Podemos modificar la dirección en la que se muestra el dropdown con dropup, dropright, dropleft; o la dirección a la que se alinean los ítems con dropdown-menu-right:

Presentación
Curriculum vitae

3. Componente NavBar

Permite construir barras de navegación más complejas que las creadas con el componente Nav. Se pueden incluir incluso cinco componentes (lototipo, texto, formulario inline, menú de navegación).

Ejemplo:

Maquetadora frontend
Inicio Servicios Contacto

4. Componente Breadcrumb

Permite mostrar en qué lugar de la jerarquía de contenidos de nuestra página nos encontramos.

Ejemplo:

Inicio
Servicios
Contacto

5. Componente Pagination

Permite navegar entre grupos de contenidos relacionados a lo largo de distintas páginas.

Ejemplo:

Anterior
Página actual
Siguiente
